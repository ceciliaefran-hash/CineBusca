# Resumo sobre HTML

## O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar páginas da web. 
Ela não é uma linguagem de programação, pois não executa lógica ou cálculos. 
Ela serve para estruturar o conteúdo de um site.

---

## Estrutura básica de um documento HTML

Um documento HTML começa com:

<!DOCTYPE html>
<html>
<head>
    <title>Título da página</title>
</head>
<body>
    Conteúdo da página
</body>
</html>

- <!DOCTYPE html> → informa que o documento é HTML5
- <html> → envolve todo o conteúdo
- <head> → contém informações da página
- <body> → contém o que aparece para o usuário

---

## Principais Tags

### <h1> até <h6>
São usadas para títulos, do maior (h1) ao menor (h6).

### <p>
Usada para criar parágrafos.

### <a>
Cria links.
Exemplo:
<a href="https://google.com">Google</a>

### <img>
Usada para inserir imagens.

---

## A importância da tag <div>

A tag <div> é usada para organizar e agrupar elementos.
Ela ajuda a estruturar o código e facilita a organização do layout da página.
