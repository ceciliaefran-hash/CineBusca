# CineBusca
🎥 Um site onde é recomendado filmes e séries de acordo com o gênero!
# Meu Projeto LabSIS
1. Requisitos Funcionais (O que o sistema faz) ⚙️

Estes são os comportamentos que mapeamos na nossa entrevista:

    RF01 - Cadastro e Login: Permitir criação de conta e acesso seguro. 🔐

    RF02 - Painel "Meu Perfil": Organização em abas (Favoritos, Quero Assistir, Já Assistido). 👤

    RF03 - Sincronização de Dados: Busca automática de informações via API externa. 🔌

    RF04 - Busca com Filtros: Pesquisa por gênero, ano, nota e plataforma. 🔍

    RF05 - Notificação de Disponibilidade: Alertar o usuário quando um filme muda de streaming. 📢

    RF06 - Compartilhamento: Gerar links para redes sociais e WhatsApp. 🔗

2. Requisitos Não Funcionais (Como o sistema opera) 🏗️

Estes requisitos garantem a qualidade e a experiência, mesmo que não sejam "botões" clicáveis. Com base no que você descreveu sobre o design moderno e escalabilidade, aqui estão os principais:

    RNF01 - Responsividade: O site deve se adaptar perfeitamente a celulares, tablets e desktops. 📱💻

    RNF02 - Usabilidade: A interface deve seguir um layout limpo e tipografia legível para facilitar a navegação. 🎨

    RNF03 - Desempenho: As buscas e o carregamento das capas de filmes devem ser rápidos para não frustrar o usuário. ⚡

    RNF04 - Escalabilidade: A arquitetura deve permitir o aumento do volume de dados e usuários sem perda de performance. 📈

    RNF05 - Disponibilidade: O sistema deve estar acessível na maior parte do tempo (ex: 99% de disponibilidade). 🌐
