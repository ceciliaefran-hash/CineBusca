# Resumo sobre CSS

## Para que serve o CSS
CSS é uma linguagem usada para deixar os sites mais bonitos e organizados. Com ele a gente consegue mudar cores, tamanhos de letras, espaçamentos e também organizar melhor os elementos da página.

Enquanto o HTML é usado para colocar o conteúdo no site (como títulos, textos e imagens), o CSS serve para cuidar da aparência. Ou seja, ele deixa a página com um visual melhor.

Usar um arquivo externo como `style.css` é o mais recomendado porque deixa o código mais organizado. Assim, o HTML fica só com o conteúdo e o CSS fica responsável pelo estilo. Além disso, o mesmo arquivo de CSS pode ser usado em várias páginas do site, o que facilita muito na hora de editar ou mudar o visual do projeto.

## Glossário de algumas propriedades do CSS

**color**  
Serve para mudar a cor do texto.
**background-color**  
Muda a cor de fundo de um elemento.
**margin**  
Define o espaço que fica por fora do elemento, separando ele de outros elementos da página.
**padding**  
Define o espaço que fica dentro do elemento, entre o conteúdo e a borda.
**display**  
Define como o elemento vai aparecer na página.
**display: flex**  
É usado para organizar melhor os elementos dentro de um container, ajudando a alinhar e distribuir eles de forma mais fácil.
**font-size**  
Usado para mudar o tamanho da letra.
**text-align**  
Serve para alinhar o texto (por exemplo: esquerda, centro ou direita).
**border**  
Serve para colocar bordas em volta de um elemento e mudar a cor, o estilo ou a espessura dessa borda.
